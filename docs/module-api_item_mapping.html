api/item_mapping

The api.item_mapping module.

Please note followings:

  • It does not guarantee that you can get the complete mapping of items
  • It provides item mappings only for PARTs
  • Item mapping may not be one-to-one

Methods

(inner) getElementID(type, item_no, color_idopt) → {Promise.<item_mapping_resource>}

Source:
  • modules/Item_mapping.js, line 17
This method returns Part-Color-Code(A.K.A ElementID) of the specified item
Examples
//Retrieves a list of element IDs of PART #1234 in any colors
getElementID("part","1234");
//Retrieves element ID of PART #1234 with color #1
getElementID("part","1234",1);
Parameters:
Name Type Attributes Description
type string The type of an item to get. Acceptable values are: PART
item_no string Identification number of an item to get
color_id number <optional>
 Color ID of an item. If not specified, API retrieves element IDs of an item in any colors.
Returns:
f successful, this method returns a item mapping resource as "data" in the response body.
Type
Promise.<item_mapping_resource>
