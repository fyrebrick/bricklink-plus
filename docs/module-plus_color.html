(inner) getInfoFromColorId(color_id) → {ColorInfo}

Source:
  • plus/Color.js, line 6
Depending in the color_id it will return the bricklink equivalent data of name, rgb en colortype (No promise)
Examples
color.getInfoFromColorId(4);
// returns {"Orange","#FF7E14",1}
color.getInfoFromColorId(53);
// returns {name:"undefined",rgb:"undefined",colortype:-1}
Parameters:
Name Type Description
color_id Number The ID of the color
Returns:
On object is given which contains the 3 data properties
Type
ColorInfo
